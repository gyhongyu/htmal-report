Specific Manufacturing Requests from Samsung

三星直接提出的代工需求

Requirement 1: USB Cable Assembly 需求一：USB 線材組裝 (USB Cable Assembly)

Samsung specifically values Foxlink's Raw Cable capability. While competitors offer basic assembly, Samsung prefers a CM (Contract Manufacturer) that can handle the complete process including raw cable manufacturing *in India*.

三星特別看中正崴的 裸線 (Raw Cable) 製造能力。雖然競爭對手能提供基礎組裝，但三星強烈傾向尋找能「在印度本地」處理包含裸線生產在內之完整工序的代工廠。

Requirement 2: SMT Flex / Multi Flex PCBA 需求二：SMT 軟板 / 多層軟板打件 (SMT Flex / Multi Flex)

Critical for their mobile phones. Currently, Samsung cannot find the right sources in India (existing sources only do basic soldering, major assembly still happens in China). They are formally asking if Foxlink can move this facility to India.

這是手機生產的關鍵零組件。目前三星在印度 找不到合格的供應商（現有印廠只能做基礎焊接，核心組裝仍需仰賴中國）。他們正式詢問正崴是否能將此產能轉移至印度。

3. The Game Changer: Import Policy Shift 參、 局勢反轉：進口關稅政策利多

Previous Roadblock Removed

過去的投資阻礙已消除

Previously, Foxlink simulated Indian production for Apple, but suspended the plan due to heavy duties on Raw Materials lacking return on investment.

過去正崴曾為 Apple 評估過印度設廠，但當時因原物料進口關稅過重，缺乏投資報酬率而暫停。

CURRENT STATUS: The Indian government now provides a DUTY BENEFIT (Zero duty for Raw Materials) if the ultimate end-product is used for mobile phones. This fundamentally changes the ROI calculation for Foxlink.

現況反轉：印度政府現已頒布關稅優惠，只要最終產品用於「手機」，其原物料即可享有【零關稅】。這從根本上改變了正崴印度建廠的 ROI (投資回報率) 試算結果。
