Cloud Path Structure

All files are automatically stored in the root directory:

所有錄音檔將自動存儲於根目錄下的：

One-Way Sync Protocol

單向備份協議

Deleting files from the phone DOES NOT delete them from the Cloud.

從手機刪除檔案 不會 刪除雲端硬碟上的備份。

Benefit: Safe to clear local storage frequently. 優勢：可隨時清理手機空間而不丟失資料。

4. Storage Strategy Conclusion

肆、 存儲策略結論

Leveraging the available Cloud Storage (e.g., 2TB plan) allows for high-fidelity recording without concern for local device limitations. The "One-Way Sync" feature acts as an automated archiving system.

利用現有的雲端大容量空間（如 2TB 方案），可進行高音質錄音而無需擔心手機本地容量限制。「單向同步」功能實際上充當了自動歸檔系統，確保資料永久保存。

Recommended Action
建議行動
Periodically verify Cloud folder integrity before bulk deleting local files.
在批量刪除手機檔案前，請定期檢查雲端資料夾的完整性。
